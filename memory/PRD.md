# Iron Beast Gym - Website PRD

## Original Problem Statement
Create a bold, energetic, and modern website for "Iron Beast Gym" in Chandrapur, Maharashtra with black/red/white theme. Mobile-friendly, fast-loading, SEO optimized for "Best Gym in Chandrapur".

## Architecture
- **Frontend**: React with Tailwind CSS, Framer Motion animations
- **Styling**: Custom dark theme with Oswald + Manrope fonts
- **Components**: Shadcn UI (buttons, inputs)
- **Backend**: FastAPI (minimal - frontend-only app)

## Core Requirements (Static)
- Hero section with headline and WhatsApp CTA
- About Us with stats (3+ years, 15+ trainers, 500+ members)
- Services (Personal Training, Weight Loss, Muscle Gain, Diet Guidance)
- Membership Plans (Monthly ₹1,500, Quarterly ₹4,000, Yearly ₹12,000)
- Transformation Gallery
- Testimonials (3 reviews)
- BMI Calculator
- Google Maps embed
- Contact section with WhatsApp integration
